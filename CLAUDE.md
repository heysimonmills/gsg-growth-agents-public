This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

`gsg-growth-agents` is an AI-powered growth/marketing agent system. Based on the configured integrations, it is intended to manage and optimize Google Ads campaigns using AI agents.

## Available Integrations (from `.env`)

The following external services are configured:

- **Anthropic API** — Claude models for agent intelligence
- **Perplexity API** — AI-powered search and information retrieval
- **Google Ads API** — Campaign and ad management (developer token, OAuth credentials, customer IDs)
- **Apify** — Web scraping and data extraction
- **GitHub** — Version control / automation integration
