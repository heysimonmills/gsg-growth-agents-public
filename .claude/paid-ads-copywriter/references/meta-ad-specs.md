# Meta Ads Specs & Best Practices

## Character Limits (Recommended — Meta previews truncate)

| Component | Preview Truncation | Hard Limit | Notes |
|---|---|---|---|
| Primary Text | ~125 chars (feed) | 500 chars | "See More" hides text after ~3 lines |
| Headline | ~40 chars | 255 chars | Shown below image/video |
| Description | ~30 chars | 255 chars | Optional — shown in some placements |

**Write for the truncation point.** The hook must land before "See More."

## Ad Formats + Placement Considerations

| Format | Primary Text Tone | Headline Role |
|---|---|---|
| Single Image | Hook-driven, story-optional | Reinforce or contrast the image |
| Video | Supports/extends the video hook | Not always shown — deprioritize |
| Carousel | Frame 1 is the hook; each card is a feature/benefit | Card headline = mini-CTA |
| Collection | Lifestyle image first; catalog below | Aspirational |
| Stories/Reels | First 3 seconds is everything | Headline less visible — nail primary text |

## Hook Principles

The hook is the first line of primary text AND the first 3 seconds of video. It must:

1. **Stop the scroll** — pattern interrupt, unexpected claim, or identity statement
2. **Create a knowledge gap** — "Here's what nobody tells you about X"
3. **Mirror the pain** — "If you've tried everything for X and nothing works..."
4. **Lead with the outcome** — "I lost 18 lbs without giving up wine. Here's how."
5. **Call out the audience** — "Attention: women over 40 with stubborn belly fat"

**Never start with the brand name.** That's for the end, not the hook.

## Primary Text Structure Options

**Problem → Agitate → Solution (PAS):**
```
[Hook — name the pain]
[Agitate — make it real, make it felt]
[Solution — introduce the product as the answer]
[Proof — stat, testimonial, or guarantee]
[CTA — reduce friction]
```

**Outcome → Mechanism → Offer:**
```
[Hook — lead with the desirable result]
[Mechanism — explain why/how it works, briefly]
[Offer — what they get, price anchor if applicable]
[CTA]
```

**Social Proof → Bridge → Offer:**
```
[Hook — lead with credibility ("10,000 customers...")]
[Bridge — connect proof to reader's situation]
[Offer + CTA]
```

## CTA Buttons (choose the lowest-friction option for funnel stage)

| Funnel Stage | Recommended CTA |
|---|---|
| Cold awareness | Learn More, Watch More |
| Consideration | Shop Now, See Menu, Get Offer |
| High intent | Buy Now, Sign Up, Get Started |
| Lead gen | Subscribe, Download, Apply Now |

## Meta Policy: Restricted Language

**Health & Wellness (most DTC clients will fall here):**
- ❌ "Cure", "treat", "heal", "fix" — implied medical claims
- ❌ Before/after framing that implies transformation ("From X to Y")
- ❌ Weight/body language: "lose X pounds", "burn fat", "slim down"
- ❌ Direct "you" statements implying personal attributes: "Your acne", "your weight"
- ✅ "Supports", "helps", "promotes", "may", "designed to"
- ✅ Outcome language focused on feeling, not measurement: "feel lighter", "look refreshed"
- ✅ Third-person social proof: "customers say", "users report"

**Financial:**
- ❌ Guaranteed returns, income claims without disclaimers
- ✅ "Potential", "up to", "some members", "results vary"

**Always flag regulated categories to the user and recommend legal/compliance review.**

## Meta Ad Strength Checklist

- [ ] Hook lands in first 125 characters
- [ ] Primary text works without the image (for feed without preview)
- [ ] Headline reinforces or contrasts the primary text (not a repeat)
- [ ] CTA button matches funnel stage
- [ ] No policy-violating language (especially health claims)
- [ ] Emojis used intentionally, not decoratively (0–2 max for most brands)
- [ ] Social proof included if available
- [ ] At least one version written for retargeting (assumes brand awareness)

## Common Meta Mistakes to Avoid

| Mistake | Fix |
|---|---|
| Opening with brand name | Open with hook — save brand for end |
| Wall of text with no breaks | Use line breaks between sections |
| Vague CTAs ("Click here") | Specific, low-friction CTAs ("Get your first box free") |
| Headline that repeats primary text | Headline should add new info or a new angle |
| Same tone for cold + retargeting | Cold = educate/hook; Retargeting = urgency/proof |
| Assuming video context for image ad | Image copy must stand alone |
